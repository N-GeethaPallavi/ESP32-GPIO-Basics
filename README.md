# ESP32 GPIO Basics

## Overview
This repository contains basic ESP32 GPIO programming examples developed as part of my Embedded Systems and Rapid Prototyping learning.

## Programs Included

### 1. Dual LED Blink
- Controls two LEDs using ESP32 GPIO pins.
- LEDs blink alternately with a fixed delay.
- Watch the hardware implementation here: https://drive.google.com/file/d/1ZvUSbAIRSDzul70IG3bJ1KChFSAY3cKt/view?usp=sharing

### 2. Push Button using Pull-Up
- Uses the ESP32 internal pull-up resistor.
- Pressing the button turns ON the built-in LED.
- Watch the hardware implementation here: https://drive.google.com/file/d/15ZO_cwHclNhknFrEXYavIM9h5a2ODiU9/view?usp=sharing

### 3. Push Button using Pull-Down
- Configured using internal pin mechanics and software-based logic.
- Pressing the button turns ON the built-in LED safely without breadboard resistors.
- Watch the hardware implementation here: https://drive.google.com/file/d/11MNPvFJw8D8Baz0fKlexlXoOBqdH-a4z/view?usp=sharing
